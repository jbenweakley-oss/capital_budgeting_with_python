## Capital Budgeting Analysis with Monte Carlo Simulation

Python model for evaluating capital expenditure decisions under uncertainty.
Built around a specialty chemicals production line scenario using industry-calibrated 
inputs (Damodaran WACC dataset, 59 firms).

Outputs: NPV, scenario analysis (bear/base/bull), tornado chart sensitivity analysis, 
and Monte Carlo simulation (10,000 iterations) producing a full probability distribution 
of outcomes.

Tools: Python, NumPy, Pandas, Matplotlib  
Concepts: DCF analysis, WACC, unlevered free cash flow, scenario modeling, 
Monte Carlo simulation, probability distributions
